Bem-vindo à Lista de Tarefas

Organize, priorize e conclua suas tarefas de forma eficiente com esta aplicação baseada em Django e Bulma.

Gerencie seu tempo e aumente sua produtividade com uma interface simples e intuitiva.

Sobre o Projeto

Este projeto foi desenvolvido para demonstrar a integração entre Django e Bulma, criando uma aplicação web dinâmica e responsiva.

Com ele, você pode gerenciar tarefas de maneira simples, mantendo-se sempre organizado.

Além disso, a interface moderna proporciona uma experiência agradável ao usuário.

  • Framework Backend: Django
  • Framework Frontend: Bulma
  • Banco de Dados: SQLite
  • Funcionalidades: CRUD completo de tarefas
